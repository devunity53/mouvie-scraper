# Guide Rapide - Mouvie Scraper\n\n## 🚀 Démarrage rapide (Windows)\n\n### 1️⃣ Installation initiale\n\n```bash\ngit clone https://github.com/devunity53/mouvie-scraper.git\ncd mouvie-scraper\npip install -r requirements.txt\n```\n\n### 2️⃣ Premier lancement\n\n```bash\npython launcher.py\n```\n\n**C'est tout!** Le launcher:\n- ✅ Vérifie la version distante\n- ✅ Télécharge les fichiers manquants\n- ✅ Met à jour les fichiers modifiés\n- ✅ Lance l'application\n\n## 🔄 Système de mise à jour automatique\n\nLe `launcher.py` compare automatiquement:\n- La version locale (VERSION.txt) avec la version GitHub\n- Le hash SHA256 de chaque fichier\n\nSi une différence est détectée, les fichiers sont téléchargés automatiquement.\n\n## 📁 Architecture modulaire\n\nL'application est divisée en modules indépendants:\n\n```\nConfig  → ConfigManager (gestion JSON)\nScraping → Scraper (BeautifulSoup)\nUI      → UIBuilder + UIComponents + ImageHandler\nMain    → MouvieApp (orchestration)\nLaunch  → launcher.py (auto-update + démarrage)\n```\n\n## ⚙️ Configuration personnalisée\n\nModifiez l'URL du site dans `config_manager.py`:\n\n```python\nDEFAULT_SITE = \"https://votresite.com/\"\n```\n\n## 🐛 Dépannage\n\n**Erreur: \"mouvie_gui.py not found\"**\n- Vérifiez que le launcher.py est dans le bon répertoire\n\n**Pas de mise à jour?**\n- Vérifiez votre connexion internet\n- Vérifiez que le repo est public\n\n**Erreur de dépendances?**\n```bash\npip install --upgrade -r requirements.txt\n```\n\n## 📝 Fichiers importants\n\n| Fichier | Rôle |\n|---------|------|\n| `launcher.py` | Script de lancement avec auto-update |\n| `VERSION.txt` | Numéro de version actuel |\n| `mouvie_gui.py` | Application principale |\n| `requirements.txt` | Dépendances Python |\n\n## 🔐 Notes de sécurité\n\n- Le launcher utilise HTTPS pour GitHub\n- Les hashes SHA256 sont utilisés pour vérifier l'intégrité\n- Aucune donnée sensible n'est stockée\n\n## 📞 Support\n\nPour les issues: https://github.com/devunity53/mouvie-scraper/issues\n"
