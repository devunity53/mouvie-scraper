# 📋 Documentation du Launcher\n\n## Qu'est-ce que le launcher?\n\nLe `launcher.py` est un script Python qui:\n1. ✅ Vérifie la version locale vs distante (GitHub)\n2. ⬇️ Télécharge les fichiers manquants ou modifiés\n3. 🚀 Lance l'application principal (mouvie_gui.py)\n\n## Comment ça marche?\n\n### 1️⃣ Vérification de la version\n\n```python\nVersion locale: 1.0.0 (lue depuis VERSION.txt)\nVersion distante: 1.0.1 (lue depuis GitHub)\n→ Une mise à jour est disponible!\n```\n\n### 2️⃣ Comparaison des fichiers\n\nChaque fichier est comparé par son hash SHA256:\n- Hash local = Fichier local\n- Hash distant = Fichier sur GitHub\n- **Si différents** → Téléchargement automatique\n\n### 3️⃣ Lancement de l'app\n\nUne fois les mises à jour complètes, l'application se lance:\n```bash\nsubprocess.run([sys.executable, \"mouvie_gui.py\"])\n```\n\n## Configuration du launcher\n\n⚠️ **IMPORTANT**: Modifiez ces variables dans `launcher.py`:\n\n```python\nREPO_OWNER = \"votre_username_github\"  # Ex: \"devun\"\nREPO_NAME = \"mouvie-scraper\"          # Nom du repo\n```\n\n## Utilisation\n\n### Lancement simple\n```bash\npython launcher.py\n```\n\n### Avec logs détaillés\nTous les logs s'affichent dans la console:\n- 🔍 Vérification des mises à jour\n- ⬇️ Téléchargements\n- ✅ Confirmations\n- ❌ Erreurs\n\n## Fichiers clés\n\n| Fichier | Rôle |\n|---------|------|\n| `VERSION.txt` | Numéro de version (ex: \"1.0.0\") |\n| `launcher.py` | Script de lancement |\n| Autres fichiers | Comparés par hash SHA256 |\n\n## Gestion des versions\n\nFormat: `MAJOR.MINOR.PATCH`\n\nExemples:\n- `1.0.0` → Version de base\n- `1.0.1` → Patch (bugfix)\n- `1.1.0` → Minor (feature)\n- `2.0.0` → Major (breaking change)\n\n## Messages du launcher\n\n### ✅ Succès\n```\n✅ 1.0.1: Vous avez la dernière version!\n✅ main.py téléchargé avec succès\n✅ 3 fichier(s) mis à jour\n```\n\n### ⚠️ Avertissements\n```\n⚠️ Impossible de vérifier les mises à jour (pas de connexion)\n```\n\n### ❌ Erreurs\n```\n❌ Erreur lors de la récupération de la version distante: [erreur]\n❌ mouvie_gui.py introuvable!\n```\n\n## Dépannage\n\n### Problem: \"impossible de vérifier les mises à jour\"\n**Solution**: Vérifiez votre connexion internet et que le repo est public\n\n### Problem: \"fichier introuvable\"\n**Solution**: Assurez-vous que le fichier existe sur GitHub\n\n### Problem: \"erreur lors du téléchargement\"\n**Solution**: Vérifiez que l'URL du repo est correcte dans launcher.py\n\n## API GitHub utilisée\n\n```\nGET /repos/{owner}/{repo}/contents/VERSION.txt\nGET /repos/{owner}/{repo}/contents\nGET /repos/{owner}/{repo}/contents/{filename}\n```\n\nL'API GitHub permet 60 requêtes par heure sans authentification.\n\n## Avantages\n\n✅ Aucune authentification requise (repo public)\n✅ Vérification d'intégrité par hash\n✅ Téléchargement sélectif (fichiers modifiés seulement)\n✅ Transparent pour l'utilisateur\n✅ Pas de dépendances supplémentaires\n\n## Limitations\n\n⚠️ Ne fonctionne que si le repo est **public**\n⚠️ Requiert une connexion internet\n⚠️ Limite de 60 requêtes/heure (API GitHub)\n\n## À venir\n\n- [ ] Authentification avec token GitHub (privé)\n- [ ] Rollback automatique en cas d'erreur\n- [ ] Mise à jour en arrière-plan\n- [ ] Fenêtre de progression\n"
